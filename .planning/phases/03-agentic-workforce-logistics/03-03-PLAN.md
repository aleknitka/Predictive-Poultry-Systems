  <files>src/predictive_poultry_systems/agents/simulation.py</files>
  <action>
    1. Create `BehavioralComponent` (inheriting from `salabim.Component`) that acts as a bridge between the Pydantic agent models and the simulation engine.
    2. Implement the `process()` loop:
       - Tick the BT: `status = self.agent_data.root_node.tick(self)`.
       - If `status == Status.RUNNING`:
         - Check for pending actions in the agent's state.
         - `yield self.hold(action_duration)` to simulate the time taken.
       - If `status == Status.SUCCESS/FAILURE`:
         - `yield self.hold(0)` to allow other components to run before next decision.
       - Always ensure a `yield self.hold()` occurs to prevent infinite simulation loops (per Pitfall 2).
    3. Support both `Customer` and `Staff` specializations.
  </action>
